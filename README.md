# Teensy Arcade Synth
3-layer polyphonic synthesizer made with Teensy using arcade buttons, encoder &amp; LCD screen all stuffed into a cigar box.

(Click the image to watch DIY Teensy Arcade Synth | Tutorial & Presets Demo)
